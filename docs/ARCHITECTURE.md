# Техническое задание
Упрощенная кибериммунная система организации воздушного движения беспилотных авиационных систем

## Назначение системы
ОрВД БАС обеспечивает:
- регистрация миссий
- авторизация полетов
- контроль воздушного пространства
- управление бесполетными зонами
- мониторинг телеметрии
- принудительное прекращение полетов (экстренные команды)
- фиксация событий безопасности

## Базовые принципы системы
- Взаимодействие между доменами безопасности происходит только через брокер сообщений (все междоменные взаимодействия происходят через брокер и проверяются монитором безопасности)
- Выдача событий безопасности по запросу.
- Передача телеметрик в сервис аналитики
- Передача событий безопасностей в сервис аналитики
- События должны записываться в журнал.
- Сертификаты будут выдаваться и подписываться Регулятором.

## Базовые функции
- Регистрация новой миссии
- Авторизация миссии
- Авторизация вылета дрона с авторизованной миссией
- Отзыв разрешения на полет / требование посадки дрона по ID
- Отображение списка запросов БВС и ответов ОрВД с указанием даты/времени запроса, идентификатора запроса, идентификатора БВС

## Расширенные функции
- Определение бесполетных зон
- Динамическое изменение бесполетных зон
- Введение локального режима "ковер" - требование посадки всем БВС в заданном районе, запрет на вылет любых БВС в районе
- Визуализация положения БВС
- Визуализация бесполетных зон
- Визуализация траектории перемещения БВС

# Дополнительно

## Безопасность
- Невозможность вылета без авторизованной миссии
- Невозможность вылета в запрещенную зону
- Невозможность выполнения миссии без необходимой лицензии
- Обеспечение журнала событий
- Аутентификация всех сообщений
- Контроль целостности между доменными взаимодействиями

## Журналирование
- дата/время события
- Идентификатор БАС
- Идентификатор миссии
- Уровень критичности
- Источник
- Результат проверки

# Архитектура 
Система будет разделена на несколько доменов безопасности

## Управляющий 
- Авторизация миссий
- Авторизация вылета
- Принятие решений об отзыве
- Проверка политик безопасности

## Контроль воздушного пространства
- Хранение бесполетных зон
- Географическая проверка маршрута
- Проверка координат БАС

## Интеграциионный
- Работа с брокером сообщений
- Прием телеметрии
- Передача событий в сервис аналитики

## Визуализация (но вроде как расширенная функция)
- Отображение БАС
- Отображение зон
- Отображение траекторий
